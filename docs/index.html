Cuatro algoritmos de clasificación compiten y colaboran para predecir el riesgo de olvido del estudiante en tiempo real.

MODELO 01 / 04
📈
Regresión Logística
Baseline · Lineal

Modelo interpretable que calcula la probabilidad de olvido mediante una función sigmoide. Utiliza 5 variables predictoras (features) y sirve como línea base para comparar los modelos más complejos. Activo en la demo del simulador.

~85%

Árbol de Decisión
No Lineal · Explainable

Divide el espacio de decisión en reglas jerárquicas tipo si/entonces. Extremadamente visual e interpretable: permite entender exactamente por qué el modelo clasifica al estudiante en riesgo. Profundidad óptima = 4.

MODELO 03 / 04

Random Forest
Ensemble · Robusto

Ensemble de 100 árboles de decisión entrenados con bagging y selección aleatoria de features. El modelo más robusto del proyecto: reduce el sobreajuste y obtiene la mayor exactitud general en validación cruzada.

SVM
Kernel RBF · Margen Máximo

Support Vector Machine con kernel RBF (Radial Basis Function). Maximiza el margen entre clases en espacios de alta dimensión. Excelente para patrones no lineales complejos como las combinaciones de latencia, repasos y nivel MCER.

⚙️ Funcionalidades

¿Qué hace
SpacedL2?

Tres módulos integrados para un aprendizaje basado en evidencia científica y predicción de datos.

🎴
Práctica Activa de Vocabulario

Tarjetas interactivas con volteo 3D. La app mide tu latencia de respuesta en milisegundos y calcula el riesgo de olvido en tiempo real tras cada interacción.

🎛️
Simulador de ML en Vivo

Ajusta las 5 variables predictoras con sliders y observa cómo cambia la predicción del modelo de Regresión Logística instantáneamente. Visualiza la curva del olvido en tiempo real.

📊
Panel de Retención

Dashboard completo con tabla de vocabulario, estado de retención por palabra, métricas agregadas y recomendación personalizada del próximo repaso según el algoritmo de espaciado.
